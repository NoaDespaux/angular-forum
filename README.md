# FrontForum

## Start the api
To start the api with the database, run:

```bash
cd pocketbase_0.28.2_linux_amd64/
./pocketbase serve
```

## Start the forum
To start the front of the application, run:

```bash
cd front-forum
npm install
ng serve
```
